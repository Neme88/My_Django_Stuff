In this course we will cover CSS selectors

CSS selectors are used to select HTML elements to style them directly even when an element is bundle into container elements like div selectors helps us to target them for styling individually

There are different types of CSS selectors which we will be listing below

  1. Tag Selector: This is used to select all elements with a specific tag
  2. Class Selector: This is used to select all elements with a specific class attribute
  3. ID Selector: This is used to select a single element with a specific id attribute
  4. Universal Selector: This is used to select all elements
  5. Descendant Selector: This is used to select all elements that are descendants of a specified element
  6. Child Selector: This is used to select all direct children of a specified element
  7. Adjacent Sibling Selector: This is used to select all elements that are adjacent siblings of a specified element
  8. General Sibling Selector: This is used to select all elements that are siblings of a specified element
  9. Attribute Selector: This is used to select elements with a specific attribute

In this course we will focus more on this four types of selectors

  • Class Selector:.container { background-color: red; }
  • ID Selector: #header { color: red; }
  • Universal selector: *{color:red}
  • Descendant Selector: p span { color: red; }
  • Child Selector: div > p { color: red; }
  • Adjacent Sibling Selector: p + p { color: red; }
  • Attribute Selector: p [attribute='value'] { color:red; }

In this course we will cover CSS Specificity

CSS Specificity is a measure of how important a rule is in determining the visual representation of an element. Specificity is determined by the order in which rules are written and the specificity of selectors. The specificity of a selector is calculated as follows:

  1. Inline styles (e.g., style="color: red;") have the highest specificity and override all other styles.
  2. ID selectors have a specificity of 1000.
  3. Class, attribute, and pseudo-class selectors have a specificity of 100.
  4. Type selectors and pseudo-elements have a specificity of 10.
  5. Universal selectors (*) have a specificity of 0.

How to check CSS Specificity

  1. Open Google Chrome
  2. Press Ctrl + Shift + I or F12
  3. Inspect the element you want to check specificity
  4. Check the specificity of the CSS rule in the Styles tab

In this course we will focus on learning the CSS BOX Model.

The CSS Box Model is a fundamental concept in CSS that defines the layout and positioning of elements on a webpage. It consists of the following components:

And box model simply means the margin, the border, the padding and the actual content

  1. Content area:This is the area where the content of the element is rendered. It can be defined by the width and height properties.
  2. Padding: This is the space between the content area and the border. It can be defined by the padding-top, padding-right, padding-bottom, and padding-left properties.
  3. Border: This is the line that surrounds the padding area. It can be defined by the border-width, border-style, and border-color properties.
  4. Margin: This is the space between the border and the adjacent elements. It can be defined by the margin-top, margin-right, margin-bottom, and margin-left properties.
  5. Box model: This is a visual representation of the CSS Box Model that helps to understand how elements are positioned and sized on a webpage.
